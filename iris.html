Visualization 1: Univariate

This visualization takes all the iris data and plots the average Petal and Sepal length of each species. With this data we can see how much Petal length differs from the Sepal length. This graph also shows how the lengths differ among each species. This visualization makes it easy to notice which Iris species, on average, has the largest lengths and which species has the smallest lengths.

This graph encodes the three different types of species included in the data set which can be seen from each bar, labeled on the x-axis. Color is used to differentiate which values correspond to which length being portrayed. The y-axis is made to fit just above to the maximum values plotted and each bar is scaled accordingly to realistically visualize each species' differences. When a viewer hovers their mouse over a bar (Petal or Sepal length as well) the measured value is shown using an added tooltip. This makes it easier to read the exact values for each bar. Also, each bar is "highlighted" when hovered over to confirm to the viewer which bar is being referenced.

Visualization 2: Bivariate

This visualization plots an Iris species' sepal width versus its sepal length to make it easy to notice if there are any correlations between the two, for any of the three included Iris species. Also included are four buttons above the graph for users to select a specific species to analyze or to compare all three together. When a button is selected, visualization 3 also changes to match the specific species or see all three species together. Viewers can also zoom in on each scatter plot to have a closer look of the data if needed. To zoom out, a viewer can double click anywhere in the box to reset the data to full size.

Color is used in this graph to encode the type of iris each data point refers to. The buttons allow users to select specific species from the data to look at instead of having to look at all three species at all times. Zooming allows for better readability and also allows users to better analyze different groups of data points. The linking between this visualization and visualization 3 also gives the viewer a better chance to understand the data per iris classification.

Visualization 3: Easy Multivariate Visualizations

This visual aims to show if specific Iris species have major differences in Petal lengths and Petal widths. This graph also makes it easy to visualize the size difference between each of the three Iris species. Viewers can hover over each bar to get more information on the data encoded. Viewers can also click on a bar and the graph will visualize that specific species as well as bring up the data of the species in visualization number 2. If the user clicks a bar again the visualization goes back to showing all three species together in both visualization 2 and 3.

Color is used to encode Petal length and petal width. Each species is grouped into sets of two bars for better comparisons. Since average value is plotted, it does not tell the full story of the data, so max and min values of each category can also be seen when a viewer hovers over a certain bar.
